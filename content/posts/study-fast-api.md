## 설치 방법

### fast api install with pipevn

`pipenv install fastapi`

### uvicorn install with pipenv

`pipenv install fastapi uvicorn`

### run server

`uvicorn main:app --reload`

### sqlalchemy 설치

`pipenv install fastapi-sqlalchemy`

## 기본 서비스

###  Swagger

http://127.0.0.1:8000/docs

### ReDoc
